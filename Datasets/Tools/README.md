# Tools for Log Analysis

This section lists recommended tools to analyze the datasets in this repository.

## Windows Tools
- Event Viewer (built-in)
- Chainsaw (fast log analysis)
- Autopsy (digital forensics)

## Linux Tools
- grep (search logs)
- awk (pattern analysis)
- less (view logs)

## Advanced Tools
- Splunk (log management)
- ELK Stack (Elasticsearch, Logstash, Kibana)
- Wireshark (network analysis)

## How to Use
1. Download datasets from Windows or Linux folders
2. Open using the appropriate tool
3. Follow README instructions in each dataset

## Purpose
These tools help students perform real-world security investigations 
using the provided datasets.

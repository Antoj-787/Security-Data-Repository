# CSI Security Data Repository

This repository provides cybersecurity datasets for students in the 
Cyber Security and Investigation (CSI) program.

## Included Datasets
- Windows Event Logs
- Linux Logs
- (Other categories added by team members)

## Purpose
To help students practice:
- Digital forensics
- Log analysis
- Threat detection

## How to Use
1. Download datasets
2. Open using recommended tools
3. Follow investigation steps in each folder

## Tools Recommended
- Event Viewer
- Wireshark
- Autopsy
- Linux command-line tools

## Project Goal
To create a centralized, cost-effective, and easy-to-use repository 
for cybersecurity learning.

Description
This dataset is a Windows 7 SP1 x64 RAM capture used in a memory forensics challenge.
It contains typical system artifacts found in a live Windows environment, such as:

Running processes

Loaded DLLs

Network connections

Command history

User activity traces

The dataset is clean and safe for academic use. It is commonly used to practice Volatility workflows and memory analysis fundamentals.

File Format
RAW memory dump

Windows 7 SP1 x64 profile compatible

Suggested Tools for Analysis
Volatility 2

Volatility 3

Strings

Timeline tools (optional)

Learning Objectives
Identify running processes and parent/child relationships

Examine network activity from memory

Explore loaded modules and handles

Understand how RAM captures preserve system state

Practice basic memory forensics techniques

Why This Dataset Is Included
This memory dump complements the disk images in the repository by providing a RAM‑based forensic dataset. It helps demonstrate how volatile memory can reveal system activity not visible on disk.

Safety Notes
This dataset is public, legal, and safe. It does not contain harmful malware or sensitive personal data.

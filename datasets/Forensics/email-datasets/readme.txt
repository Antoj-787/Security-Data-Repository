Dataset Name
NPS‑2010‑Emails – Email Encoding Test Disk Image

Source
Digital Corpora – NPS Test Disk Images
Public domain dataset created by the Naval Postgraduate School (NPS).

Description
This is a very small forensic test image designed to demonstrate how email addresses and text can be stored in different encoding formats.
The disk image contains 30 email addresses, each stored using a different encoding method.

This dataset is ideal for:

String extraction

Encoding identification

Basic artifact recovery

Demonstrating how forensic tools interpret encoded data

File Format
E01 disk image

Size: ~500 KB

Contains text files with encoded email addresses

Suggested Tools for Analysis
Autopsy / Sleuth Kit

Strings

Bulk Extractor

Hex editors

Encoding/decoding utilities

Learning Objectives
Understand how text encoding affects forensic analysis

Practice extracting strings from disk images

Compare tool output when handling encoded data

Explore how email artifacts may appear in raw disk data

Why This Dataset Is Included
This dataset adds variety to the repository by providing a lightweight, non‑image‑based artifact set. It complements the disk and memory datasets by focusing on text encoding and artifact extraction.

Safety Notes
This dataset is public, legal, and contains no sensitive or personal information.
